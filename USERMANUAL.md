# User Manual

## Smart Pesticide Assistant

This application helps farmers identify crop diseases and receive pesticide recommendations.

## Features

* Crop disease detection
* Pesticide recommendations
* Organic alternatives
* Prevention tips
* Multi-language support
* Gemini AI integration

## How to Use

### Step 1

Launch the application:

```bash
streamlit run app.py
```

### Step 2

Select your preferred language:

* English
* Telugu
* Hindi

### Step 3

Enter your Gemini API Key.

### Step 4

Upload a crop image in JPG, JPEG, or PNG format.

### Step 5

Wait for the analysis to complete.

### Step 6

Review:

* Crop name
* Disease name
* Cause
* Recommended pesticide
* Dosage
* Organic alternative
* Prevention tips

## Important Note

Always consult local agricultural experts before applying pesticides.
