# Security Policy

## Supported Versions

The latest version of Smart Pesticide Assistant is actively maintained.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly to the project maintainer.

## Security Best Practices

* Never commit API keys to the repository.
* Store secrets in environment variables.
* Use a .gitignore file to exclude sensitive files.
* Regularly rotate API keys.
* Review dependencies for known vulnerabilities.

## Gemini API Key Safety

Users must provide their own Gemini API key.

The application does not store user API keys permanently.

## Disclaimer

This project is intended for educational and research purposes. Users are responsible for verifying agricultural recommendations before use.
