# Contributing

Thank you for your interest in contributing to Smart Pesticide Assistant.

## Getting Started

1. Fork the repository.
2. Clone your fork.
3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
streamlit run app.py
```

## Contribution Guidelines

* Follow Python coding standards.
* Write clear and readable code.
* Test changes before submitting.
* Update documentation when necessary.

## Reporting Issues

If you find a bug or have a feature request, please create an issue with detailed information.

## Pull Requests

1. Create a new branch.
2. Make your changes.
3. Commit your changes.
4. Push the branch.
5. Create a Merge Request / Pull Request.
